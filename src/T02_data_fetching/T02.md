# Setup time server

- `npm start`

# Setup NextJS project

`npx create-next-app@latest`

---

# Configure import

`tsconfig.json`

```json
{
  "baseUrl": ".",
  "paths": {
    "@app/*": ["app/*"],
    "@components/*": ["components/*"]
  }
}
```

- Note, if you don't define `baseUrl`, you need to prefix the path with `./`.
